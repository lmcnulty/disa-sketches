Issues in Revising

Some of the requested changes that I'm not sure how to handle.
  • Death date
    Based on the current schema, I think this would be represented by the date of the record of death.
  • Shipping and sales information
    This is not in the schema.
  • Location: Town
    This would be easy to add, but is not in the schema: the `Location` table only has fields for Nation, State, and Colony.
  • Multiple names and name types
    Would this be for one document referring to someone by multiple names, or two different documents giving different names? In the former case, it seems like the `Actor` table would need to be expanded to hold more than one name. In the latter case, it goes to the higher level question of how we know that two documents reference the same individual, which I was told we weren't considering yet.
  • Add a document
    Again, I thought we weren't considering having multiple documents for a single individual. If this is meant to add a document independent of an individual, I would expect that it would be a completely separate form.
